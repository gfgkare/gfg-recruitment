AI Website Prompt – GFG Campus Body KARE Recruitment Website
Project Overview

Build a modern, highly interactive, visually stunning recruitment website for the GeeksforGeeks (GFG) Campus Body - KARE (Kalasalingam Academy of Research and Education).

This is NOT a normal club website.

The primary purpose is Recruitment of new members.

The design should immediately impress students and make them excited to join the club.

The website should feel like something built by a top tech company such as Google, Apple, GitHub, Stripe, Vercel, or Linear.

The design should be immersive with premium animations, futuristic graphics, glassmorphism, gradients, glowing effects, floating elements, animated backgrounds, interactive cards, and smooth page transitions.

Avoid generic templates.

Additional Requirements
Make the website feel like a premium startup landing page rather than a typical college club site.
Include placeholder areas where organizers can easily upload and update event photos, videos, and descriptions.
Ensure the "Apply Now" button is prominently visible in the navigation bar, each team card, and the final call-to-action section, all linking to the same recruitment form.
Add subtle coding-inspired decorative elements (binary streams, brackets, terminal windows, code snippets) without making the interface cluttered.
Keep animations smooth and purposeful, maintaining excellent performance and accessibility.
The overall experience should make visitors think: "I want to be part of this club."
